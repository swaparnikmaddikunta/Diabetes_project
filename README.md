# Diabetes_project
This GitHub repository presents a Jupyter notebook focused exclusively on predicting diabetes onset with Logistic Regression on the Pima Indians Diabetes dataset (768 samples, 8 features: Pregnancies, Glucose, BloodPressure, SkinThickness, Insulin, BMI, DiabetesPedigreeFunction, Age; binary Outcome: 0 = non-diabetic, 1 = diabetic).
